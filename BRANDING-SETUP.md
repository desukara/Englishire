# Englishire branding setup

All production logo and favicon files are stored in `images/`. Every HTML page now references them with relative paths, which work both at a custom domain and at a GitHub Pages project URL.

Keep the repository structure unchanged when uploading:

- `index.html`
- `style.css`
- `script.js`
- `images/`

GitHub Pages paths are case-sensitive. Do not rename `images` or any branding file without updating every HTML reference.
