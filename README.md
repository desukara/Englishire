# Englishire Production Branding Assets

This package is prepared for direct production use from the approved raster master.

## Official applications

- Website logo: crown plus ENGLISHIRE on a transparent background.
- Favicon, desktop shortcut and mobile application icon: crown only on an edge-to-edge deep royal blue field (#051942).
- No white circle or white icon background is used. Operating systems may apply their own corner mask.

## Recommended website file

Use `logo/englishire-logo-128px.webp` in the header with the included `srcset`. PNG equivalents are provided as lossless fallbacks.

## Root files

Copy the root favicon, Apple icon, Android icons, manifest and browser configuration to the website root. Copy the `icons`, `logo`, and `social` directories unchanged.

## Important source limitation

The approved artwork supplied is raster artwork, not a true vector master. These exports are correctly cleaned, cropped and sized for production web use. A genuine editable SVG/PDF would require a manual professional redraw of the crown and wordmark; this package does not mislabel an embedded raster image as vector artwork.
