プライバシー

あなたの情報は、お問い合わせと同じように慎重に考慮される必要があります。

このページでは、Englishire が受け取る可能性のある情報、それが必要な理由、およびその処理方法について説明します。

プライバシーは不必要に複雑な言葉で隠されるのではなく、明確に説明されるべきです。したがって、当社は、情報が使用される目的、情報が共有される状況、およびお客様が利用できる選択肢を以下に定めます。

最終レビュー: 2026 年 7 月 20 日

原則

情報は理由があって要求されるべきです。

Englishire は主に、学校からの問い合わせ、自己紹介、一時的な講師の代講のリクエスト、またはその他の連絡があったときに情報を受け取ります。

私たちは、問題を理解し、適切に対応し、その結果生じる職業上の関係を管理するために合理的に必要なもののみを要求するよう努めます。

情報は、フォームによって要求できるという理由だけで収集されるものではありません。

収集する前に目的を明確にする必要があります。

01

当社が受け取る可能性のある情報

お問い合わせの性質または Englishire との関係に応じて、次のような情報を受け取る場合があります。

  • お客様の氏名、役職、所属、電子メールアドレス、電話番号。
  • 学校または教育プロバイダーに関する名前、住所、ウェブサイト、最寄りの駅、およびその他の実際的な詳細。
  • 日付、時間割、学習者の年齢層、クラスの規模、レッスン形式、課題の要件。
  • あなたが送信することを選択した通信、フォーム、文書、またはその他の資料に含まれる情報。
  • 見積書、契約書、請求書発行、支払いまたは記録保持に必要な管理情報。
  • ブラウザの種類、デバイス情報、IP アドレス、基本的なアクセス記録など、Web サイトの使用に関連する技術情報。このような情報は Web サイトまたはそのサービスプロバイダーによって作成されます。

学校は、真に必要、適切かつ合法である場合を除き、学習者、保護者、教師、その他の個人に関する機密情報の送信を避けるべきです。

お問い合わせフォームは、入力された情報をフォーム処理サービスプロバイダーである Formspree に安全に送信し、お問い合わせ内容を Englishire に届けることができます。 Formspree は、独自のプライバシー規約および Englishire のフォーム アカウントに適用される設定に従って、送信内容を処理し、保持することがあります。

02

使用する理由

個人情報は次のような目的で使用される場合があります。

  • 問い合わせの受信と対応。
  • 学校、課題、時間割、または指導要件を理解する。
  • 教師の適性と空き状況を考慮する。
  • 問い合わせ、提案された業務、または確認された取り決めについて連絡する。
  • 短期講師補償の準備、管理、検討。
  • 見積書、契約書、請求書および関連記録の作成。
  • 以前の通信と任務の合理的な記録を維持する。
  • ウェブサイト、Englishire、学校、教師、その他の個人を悪用、詐欺、セキュリティ上の懸念、または違法行為から保護する。
  • 法律、規制、会計、税務上の義務を遵守する。
  • ウェブサイトとサービスの明確さ、有用性、操作性を向上させる。

当社は、個人情報を必要に応じて適切な説明または同意を得ることなく、取得目的と著しく相反する方法で利用することはありません。

03

取り扱いの基本は

Englishire は、お問い合わせ、要求されたサービス、提案または既存の取り決め、法的義務、正当な業務上の必要性、または個人によって提供された同意に合理的に関連する場合に、個人情報を取り扱います。

日本の法律により特定の通知、同意、手順または取り扱い形式が要求される場合、Englishire は該当する要件に従うよう努めるものとします。

問い合わせを送信する場合、検討されている問題に無関係な情報を提供する必要はありません。

04

情報が共有される場合

Englishire は個人情報を販売しません。

ただし、合理的に必要な場合には、情報は以下の者と共有される場合があります。

  • 関連する課題情報を必要とする教師または将来の教師。
  • 問い合わせや任務に関与する学校、教育提供者、または認定された代表者。
  • Web サイトのホスティング、フォーム、電子メール、ストレージ、管理、会計、セキュリティ、またはその他のビジネス機能をサポートするサービス プロバイダー。
  • 合理的に支援が必要な専門アドバイザー、保険会社、会計士、または法律顧問。
  • 法律により開示が要求または許可されている公的機関、規制当局、裁判所、またはその他の当事者。
  • 情報を適切に取り扱うことを条件として、合法的な組織再編、事業の譲渡または承継に関与するその他の組織。

課題のために情報が共有される場合、当社は受信者が合理的に知る必要がある内容に限定するよう努めます。

05

どれくらいの期間保管できるのか

情報は、その情報を取得した目的、関連する職業上の関係、または法律、会計、税務、セキュリティまたは記録保持の義務を満たすために合理的に必要な期間にのみ保持されます。

異なる情報が異なる期間保持される場合があります。たとえば、失敗した事前照会は、完了した譲渡または支払いに関する記録とまったく同じ方法で処理する必要はありません。

合理的に必要とされなくなった情報は、合法的な保持要件と実際的なバックアップの取り決めに従って、削除、匿名化、またはその他の方法で適切に処理される必要があります。

06

安全

Englishire は、個人情報を不正アクセス、紛失、悪用、改変、開示から保護するために、合理的な組織的および技術的措置を講じるよう努めます。

このような措置には、アクセスの制限、アカウントとパスワードの賢明な実践、安全なサービスプロバイダー、ソフトウェアの更新、記録の管理、業務情報の送信時の注意などが含まれる場合があります。

Web サイト、電子メール システム、保存方法が完全に安全であることを保証することはできません。したがって、学校は、通常の電子メールや Web フォームでの通信に不必要な機密情報を含めないようにする必要があります。

07

Cookieと技術情報

上記の確認日時点では、Web サイトのコードは Cookie を設定したり、ブラウザのストレージを使用したりしていません。これには、分析、広告トラッカー、埋め込まれたサードパーティ メディアは含まれません。

分析、埋め込みメディア、広告、またはその他の必須ではないテクノロジーが導入された場合、 クッキーポリシー それらについて説明し、必要に応じて適切な選択または同意のメカニズムを提供する必要があります。

ブラウザの設定により Cookie を拒否または削除できる場合がありますが、そうすると特定の Web サイトの機能に影響を及ぼす可能性があります。

08

日本国外でのサービス

一部の Web サイト、電子メール、ストレージ、管理、またはその他のサービスプロバイダーは、日本国外で情報を処理または保存する場合があります。

そのような場合、Englishire はプロバイダーの性質、関連する情報、転送または処理に適用される安全措置や法的要件を考慮するよう努めるものとします。

サービスプロバイダーの所在地は時間の経過とともに変化する可能性があり、情報の取り扱いにはプロバイダー独自のプライバシー規約も適用される場合があります。

09

お子様に関する情報

Englishire の Web サイトおよび問い合わせフォームは、主に学校、教育提供者、成人の代表者を対象としています。

学校は、適切な理由と法的権限がない限り、子供に関する個人情報や機密情報を提供すべきではありません。

短期講師が学習者の安全、アクセシビリティ、または教室でのサポートに関連することを真に知る必要がある場合、学校はその課題に適切かつ必要なもののみを共有する必要があります。

10

あなたの選択と要望

適用法および合理的な検証を条件として、個人は、Englishire に連絡して、自分に関する個人情報について問い合わせたり、修正、削除、制限、使用停止、またはその他の適切な形式の取り扱いを要求したりすることができます。

法的、会計、契約、セキュリティ、またはその他の正当な理由で情報を保持する必要がある場合、一部のリクエストは制限される場合があります。

当社は、個人情報を開示または変更する前に、要求を行った人の身元と権限を確認するのに十分な情報を求める場合があります。

11

このポリシーの変更

このポリシーは、ウェブサイト、サービス、テクノロジー、または法的要件が変更された場合に改訂されることがあります。

ページの上部にある日付は、ポリシーが最後にレビューされた日を示します。重要な変更は、些細な文言の中に隠すのではなく、明確に提示する必要があります。
