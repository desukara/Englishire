プライバシー

お預かりする情報も、お問い合わせと同じように慎重に扱います。

このページでは、Englishireが受け取る可能性のある情報、必要となる理由、取扱方法を説明します。

プライバシーについては、不必要に複雑な表現で隠さず、分かりやすく説明するべきだと考えています。以下に、情報を利用する目的、共有する場合、利用者が行える選択を示します。

最終確認日： 2026年7月20日

基本原則

情報を求めるときは、明確な理由が必要です。

Englishireは主に、学校からのお問い合わせ、学校の事前紹介、講師代講のご相談、その他の連絡を受けたときに情報を受け取ります。

ご相談内容を理解し、適切に回答し、その後の業務関係を管理するために合理的に必要な情報だけを求めるよう努めます。

フォーム上で質問できるという理由だけで、情報を収集することはありません。

収集する前に、目的を明確にします。

01

受け取る可能性のある情報

お問い合わせ内容やEnglishireとの関係に応じて、次のような情報を受け取る場合があります。

  • 氏名、役職、所属組織、メールアドレス、電話番号
  • 学校または教育機関の名称、住所、ウェブサイト、最寄り駅、その他の実務情報
  • 日程、時間割、学習者の年齢層、クラス人数、授業形式、ご依頼内容
  • 利用者が送付するメール、フォーム、文書、その他の資料に含まれる情報
  • 見積り、合意、請求、支払い、記録管理に必要な事務情報
  • ウェブサイトまたはそのサービス提供者が生成する場合の、ブラウザの種類、端末情報、IPアドレス、基本的なアクセス記録など、サイト利用に関する技術情報

本当に必要で、適切かつ法令上認められる場合を除き、学習者、保護者、講師、その他の個人に関する機微な情報を送らないでください。

お問い合わせフォームに入力した情報は、フォーム処理サービスのFormspreeへ安全に送信され、Englishireへ届けられます。Formspreeは、同社のプライバシー条件およびEnglishireのフォームアカウント設定に従って、送信内容を処理・保管する場合があります。

02

利用目的

個人情報は、次の目的で利用する場合があります。

  • お問い合わせの受領と回答
  • 学校、ご依頼、時間割、授業上の必要事項の把握
  • 講師の適合性と空き状況の検討
  • お問い合わせ、提案中のご依頼、確定した手配に関する連絡
  • 講師の代講・短期手配の準備、運営、確認
  • 見積書、合意書、請求書、関連記録の作成
  • 過去の連絡とご依頼に関する合理的な記録の保管
  • 不正利用、詐欺、セキュリティ上の問題、違法行為から、サイト、Englishire、学校、講師、その他の個人を守ること
  • 法令、規制、会計、税務上の義務への対応
  • ウェブサイトとサービスの分かりやすさ、有用性、運営の改善

適切な説明を行う、または必要な同意を得ることなく、取得目的と実質的に両立しない方法で個人情報を利用することは意図していません。

03

取扱いの根拠

Englishireは、お問い合わせ、依頼されたサービス、提案中または既存の手配、法的義務、正当な業務上の必要性、または本人の同意と合理的な関係がある場合に個人情報を取り扱います。

日本法により特定の通知、同意、手続き、取扱方法が求められる場合、Englishireは適用される要件に従うよう努めます。

お問い合わせ内容と関係のない情報を提供する必要はありません。

04

情報を共有する場合

Englishireは個人情報を販売しません。

ただし、合理的に必要な場合、次の相手と情報を共有することがあります。

  • 関係するご依頼情報を必要とする講師または候補講師
  • お問い合わせまたはご依頼に関係する学校、教育機関、正当な権限を持つ担当者
  • ウェブサイトのホスティング、フォーム、メール、保存、事務、会計、セキュリティ、その他の業務を支援するサービス提供者
  • 支援が合理的に必要な専門顧問、保険会社、会計士、法律顧問
  • 法令により開示が必要または認められる場合の公的機関、規制当局、裁判所、その他の関係者
  • 情報を適切に取り扱うことを条件として、正当な事業再編、譲渡、承継に関係する他の組織

ご依頼のために情報を共有する場合、受取人が合理的に知る必要のある範囲に限定するよう努めます。

05

どれくらいの期間保管できるのか

情報は、その情報を取得した目的、関連する職業上の関係、または法律、会計、税務、セキュリティまたは記録保持の義務を満たすために合理的に必要な期間にのみ保持されます。

異なる情報が異なる期間保持される場合があります。たとえば、失敗した事前照会は、完了した譲渡または支払いに関する記録とまったく同じ方法で処理する必要はありません。

合理的に必要とされなくなった情報は、合法的な保持要件と実際的なバックアップの取り決めに従って、削除、匿名化、またはその他の方法で適切に処理される必要があります。

06

安全

Englishire は、個人情報を不正アクセス、紛失、悪用、改変、開示から保護するために、合理的な組織的および技術的措置を講じるよう努めます。

このような措置には、アクセスの制限、アカウントとパスワードの賢明な実践、安全なサービスプロバイダー、ソフトウェアの更新、記録の管理、業務情報の送信時の注意などが含まれる場合があります。

Web サイト、電子メール システム、保存方法が完全に安全であることを保証することはできません。したがって、学校は、通常の電子メールや Web フォームでの通信に不必要な機密情報を含めないようにする必要があります。

07

Cookieと技術情報

上記の確認日時点では、Web サイトのコードは Cookie を設定したり、ブラウザのストレージを使用したりしていません。これには、分析、広告トラッカー、埋め込まれたサードパーティ メディアは含まれません。

分析、埋め込みメディア、広告、またはその他の必須ではないテクノロジーが導入された場合、 クッキーポリシー それらについて説明し、必要に応じて適切な選択または同意のメカニズムを提供する必要があります。

ブラウザの設定により Cookie を拒否または削除できる場合がありますが、そうすると特定の Web サイトの機能に影響を及ぼす可能性があります。

08

日本国外でのサービス

一部の Web サイト、電子メール、ストレージ、管理、またはその他のサービスプロバイダーは、日本国外で情報を処理または保存する場合があります。

そのような場合、Englishire はプロバイダーの性質、関連する情報、転送または処理に適用される安全措置や法的要件を考慮するよう努めるものとします。

サービスプロバイダーの所在地は時間の経過とともに変化する可能性があり、情報の取り扱いにはプロバイダー独自のプライバシー規約も適用される場合があります。

09

お子様に関する情報

Englishire の Web サイトおよび問い合わせフォームは、主に学校、教育提供者、成人の代表者を対象としています。

学校は、適切な理由と法的権限がない限り、子供に関する個人情報や機密情報を提供すべきではありません。

短期講師が学習者の安全、アクセシビリティ、または教室でのサポートに関連することを真に知る必要がある場合、学校はその課題に適切かつ必要なもののみを共有する必要があります。

10

あなたの選択と要望

適用法および合理的な検証を条件として、個人は、Englishire に連絡して、自分に関する個人情報について問い合わせたり、修正、削除、制限、使用停止、またはその他の適切な形式の取り扱いを要求したりすることができます。

法的、会計、契約、セキュリティ、またはその他の正当な理由で情報を保持する必要がある場合、一部のリクエストは制限される場合があります。

当社は、個人情報を開示または変更する前に、要求を行った人の身元と権限を確認するのに十分な情報を求める場合があります。

11

このポリシーの変更

このポリシーは、ウェブサイト、サービス、テクノロジー、または法的要件が変更された場合に改訂されることがあります。

ページの上部にある日付は、ポリシーが最後にレビューされた日を示します。重要な変更は、些細な文言の中に隠すのではなく、明確に提示する必要があります。
